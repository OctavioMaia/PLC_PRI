Abstract

    Foi proposta a criação de uma stylesheet, cuja função fosse a criação de diversas páginas HTML, uma para cada entrada no documento XML fornecido.

    Foi tambem proposta a criação de um servidor em JavaScript que corresse na máquina local (localhost) e que respondesse aos pedidos enviados. Estes pedidos baseiam-se no carregamento do ficheiro HTML correspondente ao respetivo URL.

    Por exemplo, ao aceder ao URL

    
            localhost:7777/1

    é carregado o ficheiro 1.html, localizado na diretoria website, utilizando para tal o seguinte código: 

    
            fs.readFile('website/'+number+'.html',function(err,data)

    Caso não encontro o ficheiro é escrito uma mensagem de erro.
